## Wall Avoidance Model

This model implements a basic obstacle avoidance strategy, allowing the vehicle to detect and avoid walls using onboard sensors. The system adjusts the motion of the robot in real time to maintain a safe distance from obstacles.

## Subsystems

### Sonars

The sonar sensors are used to measure the distance to nearby obstacles. These measurements are used as input for the control logic to detect walls and avoid collisions.
